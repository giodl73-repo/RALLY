# RALLY Wave

Use this skill to create or update RALLY waves.

## Wave requirements

- Define a product-neutral goal for shared Games Design validation
  infrastructure.
- Identify affected crates and consumer repos.
- Break the wave into small pulses.
- Include validation expectations for every pulse.
- Record non-goals that keep game-specific policy in adapters.
